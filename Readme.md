# Vanilla Web Projects 🌐

A collection of small projects and examples built with **HTML, CSS, and JavaScript** — no frameworks, just pure frontend code.  
This repository is designed as a **learning playground** and **showcase** for web development fundamentals.

---

## 📂 Repository Structure

Each project lives in its own folder with:
- `index.html` → main page
- `style.css` → styling
- `script.js` → functionality
- `assets/` → project-specific images/icons

---

## 🎯 Getting Started

Clone the repository:
```bash
git clone https://github.com/saurabh-ar/vanilla-web-projects.git
```

Navigate into the folder:
```bash
cd vanilla-web-projects
```

Open any project folder and run `index.html` in your browser.
